# DefendMe Landing Page

Landing page profesional para DefendMe - Plataforma de reclamaciones aéreas.

## 🚀 Demo

Landing page diseñada para presentación de hackathon con flujo completo de reclamación.

## ✨ Características

- 🎨 Diseño moderno y profesional tipo fintech
- 📱 Totalmente responsive
- ⚡ Animaciones suaves y fluidas
- 🔄 Flujo de 4 pasos ultra rápido
- 💎 Componentes interactivos premium
- 🎯 Colores corporativos de DefendMe

## 🛠️ Tecnologías

- React 18
- Vite
- Tailwind CSS
- Lucide React (iconos)

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview
```

## 🎯 Flujo de Usuario

1. **Paso 1**: Selección de aerolínea (+ opción "Otra")
2. **Paso 2**: Número de vuelo
3. **Paso 3**: Upload de billete/tarjeta de embarque
4. **Paso 4**: Tipo de incidencia
5. **Resultado**: Compensación estimada + seguimiento

## 💡 Notas

- Todos los rangos de compensación son dinámicos (no fijos)
- Upload de archivos funcional
- Demo visual completa para grabar presentación
- Sin backend (solo frontend demo)
